# slow-tempo
Small application for changing to tempo of songs and looping the pieces of song. Could be useful for learning.

Main stack is:
* React
* reflux
* indexedDB

## demo
wtfil.github.io/song-looper/

## install
  
    git clone git@github.com:wtfil/song-looper.git
    cd song-looper
    npm install
    npm start

## deploy
  
  npm run deploy
